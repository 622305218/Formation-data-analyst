# Formation data analyst
Le métier de data analyste (analyste de données) consiste à collecter, nettoyer et interpréter de grands ensembles de données brutes pour en extraire des informations exploitables. Il transforme ces données en insights (idées clés) pour guider les décisions stratégiques d'une entreprise
